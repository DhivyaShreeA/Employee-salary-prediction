# Employee-salary-prediction

This project is a machine learning model that predicts whether an individual earns more than â‚¹50,000 per year based on demographic and employment-related features. It is implemented in Python using Google Colab.

## Objective

To classify individuals into salary groups (`<=50K` or `>50K`) using features like age, education, occupation, and hours worked per week.


## Technologies Used

- Python
- Google Colab
- Pandas, NumPy
- Scikit-learn
- Seaborn, Matplotlib


## Dataset

- Format: CSV 
- Contains demographic and employment-related information
- Preprocessed for missing values and encoded categorical features


## Model Results

- **Model:** Random Forest Classifier
- **Accuracy:** 85.5%
- **Evaluation:** Confusion matrix and classification report
- **Visualization:** Feature importance chart
